Outras formatações

Código fonte

Podemos apresentar um código utilizando a tag <code>. Exemplo: document.getElementById('palavra')

Pré formatação

Podemos apresentar linhas de código utilizando a tag <pre> envolopando a tag <code>. Exemplo: 


num = int(imput('digite um número))
if num % 2 == 0:
    print(f'O número {num} é par!')
else
    print(f'O número {num} é ímpar!')

Citações básicas

Podemos adicionar uma citação básica utilizando a tag <q>

Citações complexas

Podemos adicionar uma citação complexa utilizando a tag <blockquote>.
Segundo Jack London, em Lobo do Mar:

O tom da última palavra contrastou de maneira tão forte com o tom suave das frases anteriores que mais pareceu uma chicotada.
