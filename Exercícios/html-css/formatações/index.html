Algumas formatações

Negrito

Formato não semântico

Podemos deixar uma frase em negrito utilizando a tag <b>

Formato semântico

Podemos deixar uma frase em negrito utilizando a tag <strong>

Itálico

Formato não semântico

Podemos deixar uma frase em itálico utilizando a tag <i>

Formato semântico

Podemos deixar uma frase em itálico utilizando a tag <em>

Marcação

Podemos marcar uma frase utilizando a tag <mark>

Tamanhos

Texto grande

Podemos criar um texto grande utilizando a tag <big>

Texto pequeno

Podemos criar um texto pequeno utilizando a tag <small>

Definições

Texto excluído

Podemos apresentar um texto excluído que deve ser lido, mas não considerado. Utilizando a tag <del>

Texto inserido

Formato não semântico

Podemos apresentar um texto incluído, mostrando que foi adicionado. Utilizando a tag <u>

Formato semântico

Podemos apresentar um texto incluído, mostrando que foi adicionado. Utilizando a tag <ins>

Texto sobrescrito

Podemos sobescrever algum termo utlizando <sup>. Exemplo: x10

Texto subscrito

Podemos subscrever algum termo utlizando <sub>. Exemplo: H2O
