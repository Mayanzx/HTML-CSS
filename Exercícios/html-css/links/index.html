Utilizando Links

Podemos acessar outras páginas por meio de links utilizando a tag <a>.

Exemplo: Acesse meu perfil do GitHub! (link que sobrepõe a página atual)

Exemplo: Acesse meu perfil do GitHub! (link que abre a página em uma nova guia)

Este comportamento diferente é resultado da adição de target="_blank" rel="external" após o link
