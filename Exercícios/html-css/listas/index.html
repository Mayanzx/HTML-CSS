Desenvolvendo listas

Listas ordenadas

podemos criar listas ordenadas utilizando a tag <ol> e usando a tag <li> para cada item adicionado. Como exemplo, irei montar minha rotina:

  1. Acordar
  2. Tomar café da manhã
  3. Ir para academia
  4. Almoçar
  5. Lavar a louça
  6. Estudar
  7. Tomar café da tarde
  8. Faculdade
  9. Jantar
  10. Dormir

Listas não ordenadas

podemos criar listas não ordenadas utilizando a tag <ul> e usando a tag <li> para cada item adicionado. Como exemplo, irei montar uma lista de compras:

  • Frutas
  • Produtos de limpeza
  • Bebidas
  • Doces
  • Verduras
  • Roupas
  • Remédios
